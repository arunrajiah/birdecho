## What this changes

Adds a one-line mention of Perch — an open-source mobile companion app for BirdNET-Go stations — to the Related Projects or Community Tools section of the README.

## Proposed addition

```markdown
- **[Perch](https://github.com/arunrajiah/perch)** — open-source Android/iOS companion app.
  Connects to your BirdNET-Go station via the BirdWeather API. Live detection feed, audio
  playback, species browser, 14-day chart, and local notifications for favourite species.
  MIT licensed.
```

## Notes

- Perch uses the BirdWeather API, so it requires the user to have BirdWeather integration enabled on their BirdNET-Go instance.
- It is not affiliated with or endorsed by the BirdNET-Go project.
- Happy to adjust wording or placement to match the project's conventions.
